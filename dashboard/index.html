วิธีตั้งค่าอุปกรณ์ Pressure Sensor

1

เปิดเครื่อง

เสียบปลั๊ก และ ทำการกดสวิตช์

💡 LED สีเขียวจะสว่าง แสดงว่าอุปกรณ์พร้อมทำงาน
2

หากไม่มีรหัส Wi-Fi บันทึกอยู่ — บอร์ดจะปล่อยสัญญาณ Wi-Fi ออกมา

หยิบมือถือขึ้นมาเปิดหน้า ตั้งค่า Wi-Fi แล้วค้นหาสัญญาณที่ชื่อ:

PressureSensor_Setup

รหัสผ่าน: 12345678

3

หน้าเว็บตั้งค่าจะเด้งขึ้นมาอัตโนมัติ

หลังจากเชื่อมต่อกับ PressureSensor_Setup แล้ว หน้าเว็บตั้งค่าจะเปิดขึ้นเองทันที (เหมือน Wi-Fi โรงแรม) หากไม่เด้ง ให้เปิดเบราว์เซอร์แล้วพิมพ์ 192.168.4.1

4

เลือก Wi-Fi แล้วกรอกรหัสผ่าน

กดปุ่ม "Configure WiFi" → เลือกชื่อ Wi-Fi ของคุณจากรายการ → กรอกรหัสผ่าน → กด Save

💡 LED สีฟ้าจะสว่างและอุปกรณ์จะจำรหัสผ่านนี้ไว้ในชิป ไม่ต้องตั้งค่าใหม่ทุกครั้ง
5

บอร์ดรีสตาร์ทและเชื่อมต่อ Wi-Fi ใหม่

ESP32 จะ Restart ตัวเองอัตโนมัติ เชื่อมต่อ Wi-Fi ที่กำหนด แล้วส่งข้อมูลเข้า Firebase ได้ทันที หน้า Dashboard นี้จะแสดงสถานะ Online ภายใน 30 วินาที

🔄

หากต้องการเปลี่ยน Wi-Fi ใหม่?

ให้เปิดกล้องและกดปุ่ม BOOT (ปุ่มบนบอร์ด ESP32 ฝั่งซ้ายมือ ใกล้พอร์ต USB) ค้างไว้ 3 วินาที บอร์ดจะลืมรหัส Wi-Fi เก่า แล้วกลับไปทำซ้ำตั้งแต่ข้อ 2
